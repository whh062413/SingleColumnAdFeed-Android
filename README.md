## 功能

- 单列广告卡片信息流（大图/小图/视频）
- 分类切换（推荐 / 电商 / 本地）
- AI 智能广告分析（摘要 + 标签生成）
- AI 搜索关键词提取
- 下拉刷新 + 骨架屏加载
- 点赞/收藏（SQLite 持久化）
- 曝光 & 点击埋点

## 技术栈

Kotlin + Java • Jetpack Compose • Material 3 • MVVM • OkHttp • SQLite • OpenAI API
